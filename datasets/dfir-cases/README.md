# Datasets

This directory contains datasets used for testing and demonstration purposes within the Python Security Toolkit.

All datasets included in this repository are **synthetic, anonymized, or sanitized** and are intended strictly for educational and research purposes.

No real-world sensitive information, personal data, credentials, or confidential logs are included in this repository.

Some datasets may simulate:

- system logs
- authentication events
- network traffic
- security alerts
- Indicators of Compromise (IOCs)

These datasets are designed to support the development and testing of DFIR, threat intelligence, and cybersecurity automation tools.

If you use real datasets for experimentation, ensure that **all sensitive information is properly anonymized before committing files to version control.**
